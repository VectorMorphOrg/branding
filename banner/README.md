# Banners

This directory contains the official promotional banners for the
VectorMorph project.

Banners are intended for use on:

- GitHub repositories
- Discord
- X (Twitter)
- Websites
- Documentation
- Presentations
- Community media

## Current Assets

### VectorMorphBanner.png

Primary promotional banner featuring:

- VectorMorph branding
- VMR-01 — Dr. Evelyn Vector (E.V.E.)
- Adaptive Execution Graph Proof-of-Work (AEG-PoW)
- EvoGraph
- Official VectorMorph visual identity

## Usage

These banners may be shared by the community when referencing,
discussing, or promoting the VectorMorph project.

Please do not modify official artwork in a manner that could
misrepresent the project or imply official endorsement of
third-party products or services.

For official logos, icons, and character artwork, see the
other directories within this repository.
